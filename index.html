Project

DataBase To Data WareHouse

Re-engineer Database to Data Warehouse

The main objective of the project is to migrate the data from one database to another database. The source database is provied we need to perform ETL process and transfer the data to target database.

The source database is the Northwind database built by Microsoft.The Source database is created by using MySQL.The data is taken from My SQL and stored in Python Pandas.In Python Pandas ETL process is taking place.

ETL is a Data Integration process that combines data from multiple data sources into a single,consistent store that is loaded into a data warehouse or target System.

The target database we use is Postgre sql. 

                        The steps involved are 
                        i) Create source table using MySQl.
                        ii) Export the tables to CSV files.
                        iii) Draw the target schema and decide on the modifications that need to be done.
                        iv) Data cleaning,data wrangling and data analytics done in the CSV file using python panda.
                        v) Fresh CSV files are created and datas are imported to the target table.
